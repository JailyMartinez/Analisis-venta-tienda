# Análisis de Ventas - Tienda de Conveniencia

## Descripción
Análisis exploratorio de datos de ventas utilizando Python y Pandas.

## Objetivo
Este proyecto demuestra habilidades en:
- Manipulación de datos con Pandas
- Visualización con Matplotlib
- Análisis exploratorio de datos (EDA)

## Tecnologías
- Python 3.x
- Pandas
- Matplotlib
- NumPy
